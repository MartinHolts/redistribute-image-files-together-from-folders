# redistribute-image-files-together-from-folders

🛠️ Build Instructions: Script to EXE
This guide explains how to convert the Python script into a standalone .exe file. This allows you to run the image consolidator on any Windows machine without needing Python installed.
📋 Prerequisites
Python Installed: Ensure Python is installed on your system.
Script Name: Ensure your script is named consolidate_images.py.
